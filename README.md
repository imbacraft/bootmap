# bootmap.dev

The Java ecosystem reference, reorganized and deeply explained. Production patterns, version-aware, cross-linked.

## What this is

bootmap.dev reorganizes official Java ecosystem documentation into a navigable, deeply-explained format with production code examples drawn from real shipped systems. Content is licensed under Apache 2.0 (derivative of upstream documentation) with full attribution.

## Stack

- [Astro](https://astro.build) with the [Starlight](https://starlight.astro.build) documentation theme
- MDX for content; React for interactive components

## Local development

```bash
npm install        # install dependencies
npm run dev        # start dev server at http://localhost:4321
npm run build      # produce static site in dist/
npm run preview    # serve built site locally
```
